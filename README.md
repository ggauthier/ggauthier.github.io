***ISO/IEC 4932*** 

This repository contains the JSON-LD files that make up the figures of AnneFx B

Files B.6_ggauthier.jsonld, B.8_ggauthier.jsonld and B.10_ggauthier.jsonld are versions of B6, B8 and B10 respectively, modified so as to use the JSON-LD contexts afa_en.jsonld and afa_fr.jsonld from working URLs.
